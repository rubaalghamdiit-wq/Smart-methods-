# Quadruped Robot Dog 🐕🤖

## Project Overview
This project presents the initial mechanical design of a quadruped robotic dog developed for the Smart Methods training assignment.

## Design Features
- Four-legged robot design.
- Two joints per leg.
- Servo motors used for movement.
- Central battery and controller placement for better balance.

## Expected Challenges
- Maintaining stability while walking.
- Joint friction.
- Possible leg slipping.
